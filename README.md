# Social Media Graphics

Hosted graphics for Untitled Publishers social media posts.

## Folder Structure

```
/week-2026-01-11/
  FB_TheHotMess_Day1_Slot1.png
  FB_TheHotMess_Day1_Slot2.png
  FB_PerpetualRookie_Day1_Slot1.png
  ...
  IG_Day1_Slot1.png
  IG_Day1_Slot2.png
  ...
```

## Usage

Images are served via GitHub Pages at:
```
https://mrmicaiah.github.io/social-graphics/week-YYYY-MM-DD/FILENAME.png
```

These URLs are auto-generated by the Social Media Scheduler v7 and included in the Metricool CSV.

## Weekly Workflow

1. Generate schedule in the scheduler
2. Download Canva Bulk CSV
3. Create graphics in Canva using Bulk Create
4. Export PNGs with exact filenames
5. Upload to this repo in the correct week folder
6. Upload Metricool CSV - images auto-attach!
